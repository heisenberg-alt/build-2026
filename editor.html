How this works

What this is. A browser editor for the three Microsoft Build 2026 customer decks. The preview uses the same Marp renderer that produces the live HTML on the site.

Editing. Pick a deck, edit the markdown on the left, watch the preview update on the right. Changes auto-save to this browser (localStorage) — they survive a refresh but are local to you.

Get a PPTX out — fully automated.

  1. Edit the markdown here.
  2. Click Commit on GitHub →. The GitHub web editor opens with your file pre-loaded; paste the content (already in your clipboard) and commit to main.
  3. The build-decks.yml Action runs (~1 min), builds the .pptx and .pdf, and attaches them to the Latest PPTX release.
  4. Click ⬇ Latest PPTX to grab the freshly built file.

Local alternative. Click Download .md, then run marp file.md --pptx on your own machine.

Reset reloads the canonical version from the repo and clears your local edits for that deck.
